# Tim Gelens — Kinesitherapie (demo-website)

Een zelfstandige demo-website voor kinesitherapiepraktijk Tim Gelens.
Alles zit in deze ene map en werkt zonder internet of installatie.

## Bestanden

| Bestand      | Beschrijving                                  |
|--------------|-----------------------------------------------|
| `index.html` | De volledige pagina (structuur + inhoud)      |
| `style.css`  | Vormgeving en responsive layout               |
| `script.js`  | Kleine interacties (mobiel menu, contactform) |

## Bekijken

Dubbelklik op `index.html` — de site opent in je browser. Klaar.

## Online zetten (gratis)

De map bevat enkel statische bestanden, dus je kan ze gratis hosten:

- **GitHub Pages**: zet de map in een repository en activeer Pages.
- **Netlify / Vercel**: sleep de map naar het dashboard.

## Aanpassen

- Teksten, adres en openingsuren staan in `index.html`.
- Kleuren pas je bovenaan `style.css` aan (de `--teal-*` variabelen).
- Vervang telefoonnummer, e-mail en RIZIV-nummer door de echte gegevens.

> Dit is een demo. Het contactformulier verstuurt nog geen echte e-mail —
> koppel het aan een formulierdienst (bv. Formspree) om berichten te ontvangen.
